# Aether


Adaptive Assistive Navigation System for Outdoor Safety

This project implements a mobile navigation framework for individuals with cognitive, sensory, or mobility impairments. Features include energy-efficient GPS tracking, dynamic route learning, wearable integration (e.g., OMI necklace), AI-assisted environmental awareness, emergency alerts, and behavioral analytics to enhance user independence and safety.
